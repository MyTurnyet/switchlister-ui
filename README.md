# Switchlister UI

This is a UI for a model railroad switch list API.

### Explanation

Switch Lists are a list of pick-ups and set outs for a train as it travels along its route.

#### Example:

Train Name: Local North #123

| Station | Industry  | Pick Up   | Set Out |
|---------|-----------|-----------|---------|
| Monroe  |           |           |         |
|         | Brewery   |           |         |
|         |           | BNSF 1234 | CP 4567 |
|         |           | BNSF 9876 |         |
|         | Feed Mill |           |         |
|         |           |           | BN 9999 |
|         |           |           | BN 8833 |
